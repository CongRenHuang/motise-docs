16:11 Hunag Con Ren國際外包市場常見「一年期Retainer合約」大致內容

1. 報酬結構
- 月費制（Retainer），每月固定支付，不論當月需求量
- 全端（React Native + Node.js）資深外包行情：約 $1,500–$3,000+ USD/月（對應每月約80-160小時保留產能）
- 12個月 = 至少 $18,000 USD起，不論期間實際開發量多寡

2. 付款義務
- 月費為「保留開發產能」的費用，非按件計酬——即使某月沒有新需求，仍須支付
- 通常需預付或月初支付，非完成後付款

3. 終止條款
- 書面通知期：常見30-60天（非14天）
- 提前終止：通知期內月費仍需支付（buy-out），部分合約另收termination fee（如1-2個月費用）
- 「無賠償解約」通常指「通知期滿後不續約即可」，而非「隨時可零成本退出已承諾的月費」

4. 排他性
- 部分retainer會要求一定比例工時排他（不可同時承接其他同等規模專案）

5. 觀察/磨合期
- 若有，通常1-3個月，期間費率較低或工時較少；之後即恢復全額月費——不是整個合約期都打折

6. IP/所有權
- 隨月費支付同步移轉，或合約終止時整批移轉，不像SOW逐階段分批
