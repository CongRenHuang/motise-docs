## B-4 延遲條款說明

**不列入開發方延遲責任之情況：**

- App Store / Google Play 審核等待或平台退件期間
- 客戶未如期提供必要資源、帳號或存取權限所造成之延誤
- Stripe KYC 驗證未如期完成所造成之付款功能延遲
- Apple Developer 組織帳號 D-U-N-S Number 核發延誤所造成之送審日期順延
- 不可抗力事件（天災、平台政策突變等）

**延遲賠償條件：**

若因開發方原因導致功能未完成，且里程碑時間超過 14 個日曆天，客戶方得依下列公式計算賠償。賠償僅於軌道 C 正式交付驗收時一次計算，不逐軌道累計：

```
賠償金額 = 已支付款項 × (未完成驗收項目數 ÷ 附件 C 總驗收項目數) × 50%
賠償上限 = 已支付款項之 20%
```
